# Career Passport
## 東京大学 ブロックチェーンイノベーション寄附講座  
### MOFグループリポジトリ

---

## フロントエンド（React）

### パッケージ復元
```bash
cd frontend
npm install
```

### サーバー起動
```bash
cd frontend
npm start
npm run dev
```

---

## バックエンド（Node.js）

### パッケージ復元
```bash
cd backend
npm install
```

### サーバー起動
```bash
cd backend
npm start
# または
node app.js
npm run dev
```
