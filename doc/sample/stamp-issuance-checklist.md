# スタンプ発行チェックリスト

このドキュメントは、デモ用データのスタンプ発行を効率的に行うためのチェックリストです。

## 発行順序

各学生が異なる NFT、一部同じ NFT をバラバラで持つように、以下の構成でスタンプを発行します：

- **タナカ タロウ**: テックイノベーション（3 個）、マーケティングプロ（3 個）→ 2 つの NFT 発行可能
- **ヤマダ ハナコ**: マーケティングプロ（3 個）、クリエイティブスタジオ（3 個）→ 2 つの NFT 発行可能
- **サトウ ジロウ**: テックイノベーション（3 個）、クリエイティブスタジオ（3 個）→ 2 つの NFT 発行可能

これにより：

- タナカとサトウがテックイノベーションから NFT（共通）
- タナカとヤマダがマーケティングプロから NFT（共通）
- ヤマダとサトウがクリエイティブスタジオから NFT（共通）

## 発行後の確認事項

### 各学生のスタンプ数確認

各学生が以下のスタンプ数を持っていることを確認してください：

- **タナカ タロウ**: 6 個（テックイノベーション 3 個、マーケティングプロ 3 個）
- **ヤマダ ハナコ**: 6 個（マーケティングプロ 3 個、クリエイティブスタジオ 3 個）
- **サトウ ジロウ**: 6 個（テックイノベーション 3 個、クリエイティブスタジオ 3 個）

### NFT 証明書発行可能性の確認

各学生は、特定の企業から 3 個のスタンプを持っているため、以下の NFT 証明書を発行できます：

- **タナカ タロウ**:

  - ✅ テックイノベーション株式会社から Common NFT（同一企業 3 個）※ サトウと共通
  - ✅ マーケティングプロ株式会社から Common NFT（同一企業 3 個）※ ヤマダと共通
  - ❌ クリエイティブスタジオ株式会社からは発行不可（スタンプなし）

- **ヤマダ ハナコ**:

  - ❌ テックイノベーション株式会社からは発行不可（スタンプなし）
  - ✅ マーケティングプロ株式会社から Common NFT（同一企業 3 個）※ タナカと共通
  - ✅ クリエイティブスタジオ株式会社から Common NFT（同一企業 3 個）※ サトウと共通

- **サトウ ジロウ**:
  - ✅ テックイノベーション株式会社から Common NFT（同一企業 3 個）※ タナカと共通
  - ❌ マーケティングプロ株式会社からは発行不可（スタンプなし）
  - ✅ クリエイティブスタジオ株式会社から Common NFT（同一企業 3 個）※ ヤマダと共通

### 異業種レア NFT 発行可能性の確認

各学生は、異なるカテゴリのスタンプを持っているため、レア NFT も発行できます：

- **タナカ タロウ**:

  - カテゴリ: `programming`（3 個）、`marketing`（3 個）
  - ❌ 異業種 3 種類でレア NFT 発行不可（2 種類のみ）

- **ヤマダ ハナコ**:

  - カテゴリ: `marketing`（4 個）、`business`（2 個）、`design`（3 個）
  - ✅ 異業種 3 種類でレア NFT 発行可能

- **サトウ ジロウ**:
  - カテゴリ: `programming`（3 個）、`design`（3 個）
  - ❌ 異業種 3 種類でレア NFT 発行不可（2 種類のみ）

**注意**: サトウがレア NFT 発行可能にするには、クリエイティブスタジオからのスタンプのカテゴリを`design`以外（例: `marketing`や`business`）に変更するか、追加のスタンプを発行する必要があります。現在の構成では、サトウは 2 種類のカテゴリのみを持っているため、レア NFT 発行はできません。ただし、この構成により、各学生が異なる組み合わせの NFT を持ち、一部は共通の NFT を持つというデモの目的は達成されています。

## 発行手順

1. 企業のウォレットアドレスで MetaMask に接続
2. `/org/stamp-issuance`ページにアクセス
3. 上記のチェックリストを参考に、各学生にスタンプを発行（各学生は 2 企業から 3 個ずつ、合計 6 個）
4. 発行後、学生のウォレットアドレスで`/student/mypage`にアクセスしてスタンプを確認
5. NFT 証明書発行可能性を確認

---

## 注意事項

- ウォレットアドレスは実際のデモ環境で使用する場合は、Anvil のテストアカウントを使用してください
- スタンプの発行には、企業のウォレットアドレスで MetaMask に接続する必要があります
- 各学生は 2 企業から 3 個ずつスタンプを受け取り、それぞれ異なる組み合わせの NFT 証明書を取得できます
- 一部の学生は同じ企業から NFT 証明書を取得するため、共通の NFT を持つことになります
