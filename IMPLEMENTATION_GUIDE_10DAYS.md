## 概要

この手順書は、**AI を活用して 10 日で実装**する CareerPassport アプリの実装ガイドです。

**実装戦略**:

- 🚀 **UI ファースト**: 先に UI を完成させ、モックデータで動作確認
- 🔗 **後から統合**: UI 完成後にブロックチェーン連携を追加
- 🤖 **AI 活用**: コード生成、デバッグ、最適化に AI を最大限活用
- 🔐 **Web3 設計**: WalletConnect 認証、VC 設計、プライバシー保護

**実装範囲**:

- ✅ フロントエンド UI（ユーザー・企業向け、全ページ実装完了）
- ✅ WalletConnect 統合（MetaMask 認証）
- ✅ VC（Verifiable Credential）設計への移行
- ✅ メッセージ UI 改善（個人情報非表示設計）
- ✅ ローカルストレージでのデータ管理（一時的）
- ⏳ スマートコントラクト（NFT 発行）
- ⏳ バックエンド API 実装
- ⏳ ブロックチェーン連携
